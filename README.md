# CaliRunner

- CaliRunner es un videojuego de tipo runner inspirado en la ciudad de Cali, Colombia que es de donde soy. Para nadie es un secreto que una de las problematicas mas grandes de la ciudad es el estado de las calles. En este juego quise plasmar esa realidad. Vas en la perspectiva de una moto y debes esquivar los huecos y conos de obras mientras recorres las calles caleñas.

# Como se juega?
- Flecha arriba / Flecha abajo - Para cambio de carriles
- Espacio - Iniciar el Juego
- R - Reiniciar el juego

# Obstaculos
- Huecos - el que seria el obstaculo principal
- Conos - otro obstaculo agregado en el juego

# Mecanica de llantas
- Agregue una mecanica con unas llantas el cual si chocas con un obstaculo y tienes llantas recogidas, gastas una llanta y no se te pincha la moto.

# Dificultad
- La velocidad del juego va aumentando segun las cuadras recorridas que tengas en el juego, cada 5 segundos vas a aumentar 1 cuadra.
